## üìù NOTES & BEST PRACTICES

### Compliance Tips
1. Set calendar reminders 90 days before deadlines
2. Maintain a compliance binder with all filings
3. Keep copies of all filed documents
4. Document all meetings with minutes
5. Review calendar quarterly for updates
6. Coordinate with professional advisors
7. Budget for all fees and expenses

### Common Pitfalls to Avoid
- Missing filing deadlines (penalties apply)
- Forgetting to update registered agent
- Not holding required meetings
- Commingling entity funds
- Inadequate record keeping
- Missing estimated tax payments
- Letting licenses lapse

### Annual Review Process
1. Review all deadlines in December
2. Update calendar for new year
3. Confirm all professional relationships
4. Budget for upcoming year's fees
5. Schedule all quarterly meetings
6. Set up reminder system
7. Assign responsibilities
